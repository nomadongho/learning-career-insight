# Learning & Career Insight

A static website with three self-assessments to help you understand your learning style and career decision-making.

## Assessments

| # | Title | File |
|---|-------|------|
| 1 | **Learning Styles** – Rank four descriptions per row to reveal your learning profile across CE, RO, AC, and AE dimensions. | `index.html` |
| 2 | **Surface vs Deep Learners** – Answer 20 statements (A–E) to see whether your current study approach is surface-level or deep. | `surface-deep.html` |
| 3 | **3 Factors of Career Choice** – Rate 33 items on importance and influence scales to profile what drives your career decisions. | `career-choice.html` |

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Assessment 1: Learning Styles |
| `surface-deep.html` | Assessment 2: Surface vs Deep Learners |
| `career-choice.html` | Assessment 3: 3 Factors of Career Choice |
| `history.html` | Saved results for Assessment 1 |
| `surface-deep-history.html` | Saved results for Assessment 2 |
| `career-choice-history.html` | Saved results for Assessment 3 |

## Usage

Open any HTML file directly in your browser — no build step or server required.

```
open index.html
```

Results are saved automatically to the browser's `localStorage`.
