# test

This repository contains a basic static website.

Open `index.html` in your browser to use the assessment locally.
Open `history.html` to view saved test result history.
Open `surface-deep.html` for the Surface vs Deep Learners test.
Open `surface-deep-history.html` to view saved Surface vs Deep results.
Open `career-choice.html` for the 3 Factors of Career Choice test.
Open `career-choice-history.html` to view saved Career Choice results.
